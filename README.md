# Fuzone-LM
LLM app for School. 
学校をはじめとした、複雑なAIアプリに馴染みのない人でも迷わず簡単に使える教育向けAIアシスタント。

---

## 🌟 プロジェクトの概要

**Fuzone-LM** は、「誰でも迷わず、親しみを持って使えること」を最優先に設計されたAIアプリケーションです。
学校などの現場において、生徒や教職員が日常の「バディ（相棒）」として自然にAIに触れ、学習や活動に活かせる環境を提供します。

## ✨ 特徴とこだわり

### 1. 圧倒的にシンプルなユーザー体験 (UX)
* **名前だけの超簡単サインアップ:** 煩わしい二段階認証などの手続きは一切不要。ユーザー名を入力するだけで、すぐに自分のアカウントを作成して使い始めることができます。
* **直感的で豊富なUI:** AIアプリに慣れていない人でも、直感的に操作できるデザインと豊富なUIを備えています。

### 2. あなたを覚える「バディAI」機能
* **Memory（記憶）システム:** AIはアカウントごとに最大300字程度の「Memory」を持っています。
* **性格や特徴の記憶:** ユーザーのちょっとした性格や好みをAIが記憶・蓄積。使うほどに「自分だけのバディ」として馴染み深い存在へと成長していきます。

### 3. 高性能かつ安全なシステム設計
* **超高速・高性能なLLMの採用:** Groq APIを経由し、非常に高性能なオープンソース言語モデルである **「GPT OSS 120B」** を使用。スムーズで的確な対話を実現します。
* **GAS（Google Apps Script）による安全な鍵管理:** APIキーやパスコードなどの重要なデータは、GASと連携させることで安全に保管・保護されています。

## 🛠️ 技術スタック

* **LLM API:** Groq API（Model: GPT OSS 120B）
* **Backend / Security / Key Management:** Google Apps Script (GAS)
